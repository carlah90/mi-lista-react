# React + Vite
# Mi lista React  - Aplicacion de tareas en React 

Aplicacion ha sido desarrollada como parte de evaluacion de Front end, utilizando **React + vite**, 
esta aplicacion permite agregar tareas, marcarlas como hechas, eliminarlas, ademas de tener un contador que indica
cuantas tareas quedan pendientes de hacer y tambien frases motivacionales que cambian con cada tarea agregada o al cambiar de tema de claro a oscuro. Fue diseñada en tonos pastel y cuenta con un boton ubicado en la parte superior 
derecha para el cambio de tema. 

# Caracteristicas principales
 
 * Agregar tareas nuevas 
 * Marcar tareas completadas
 * Eliminar tareas no deseadas 
 * Modo claro y modo oscuro
 * Frases motivacionales
 * Banner en el header
 * Favicon 
 * Colores personalizados en tonos pastel.

# Cómo ejecutar el proyecto


1. Clonar el repositorio:
       En el bash
      git clone https://github.com/carlah90/mi-lista-react.git

2. Instalas dependencias 
    En el terminal (en caso de no tenerla)
         npm install 

3. Ejecutar en terminal 
    Npm run dev         

4. Abrir en el navegador
